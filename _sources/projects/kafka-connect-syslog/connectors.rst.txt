================
Syslog Connector
================

The Syslog Connector provides a Kafka Connect :term:`Source Connector` that can receive data from remote hosts using the
syslog protocol. Syslog messages over UDP per :rfc:`5426`, TCP per :rfc:`6587`, and TCP with SSL per :rfc:`5425`.
