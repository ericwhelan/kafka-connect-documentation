=========
SyslogKey
=========

This schema represents the key that is written to Kafka for syslog data. This will ensure that all data for a host ends up in the same partition.

.. csv-table:: SyslogKey
    :header: "Name", "Type", "Optional", "Default Value", "Documentation"
    :widths: auto

    "remote_address",":ref:`schema-String`","false","","The ip address of the host that sent the syslog message."
