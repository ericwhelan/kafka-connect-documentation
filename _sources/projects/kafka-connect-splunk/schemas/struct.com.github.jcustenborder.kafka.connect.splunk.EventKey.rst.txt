========
EventKey
========

This schema represents the key for the data received from the Splunk listener.

.. csv-table:: EventKey
    :header: "Name", "Type", "Optional", "Default Value", "Documentation"
    :widths: auto

    "host",":ref:`schema-String`","false","","The host value to assign to the event data. This is typically the hostname of the client from which you're sending data."
