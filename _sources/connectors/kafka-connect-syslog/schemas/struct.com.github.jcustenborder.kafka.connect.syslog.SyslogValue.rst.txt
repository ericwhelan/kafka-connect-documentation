=========================================================
com.github.jcustenborder.kafka.connect.syslog.SyslogValue
=========================================================

This schema represents a syslog message that is written to Kafka.


+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| Name           | Optional | Schema                                             | Default Value | Documentation                                                                                |
+================+==========+====================================================+===============+==============================================================================================+
| date           | true     | :ref:`schema-int64`                                |               | The timestamp of the message.                                                                |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| facility       | true     | :ref:`schema-int32`                                |               | The facility of the message.                                                                 |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| host           | true     | :ref:`schema-string`                               |               | The host of the message.                                                                     |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| level          | true     | :ref:`schema-int32`                                |               | The level of the syslog message as defined by [rfc5424](https://tools.ietf.org/html/rfc5424) |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| message        | true     | :ref:`schema-string`                               |               | The text for the message.                                                                    |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| charset        | true     | :ref:`schema-string`                               |               | The character set of the message.                                                            |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| remote_address | true     | :ref:`schema-string`                               |               | The ip address of the host that sent the syslog message.                                     |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
| hostname       | true     | :ref:`schema-string`                               |               | The reverse DNS of the `remote_address` field.                                               |
+----------------+----------+----------------------------------------------------+---------------+----------------------------------------------------------------------------------------------+
