===========
Destination
===========

This schema is used to represent a JMS Destination. Typically a [Queue](http://docs.oracle.com/javaee/6/api/javax/jms/Queue.html) or a [Topic](http://docs.oracle.com/javaee/6/api/javax/jms/Topic.html)

.. csv-table:: Destination
    :header: "Name", "Type", "Optional", "Default Value", "Documentation"
    :widths: auto

    "destinationType",":ref:`schema-String`","false","","The type of JMS Destination. Typically Queue, Topic."
    "name",":ref:`schema-String`","false","","The name of the destination. This will be the value of `Queue.getQueueName()](http://docs.oracle.com/javaee/6/api/javax/jms/Queue.html#getQueueName()) or [Topic.getTopicName() <http://docs.oracle.com/javaee/6/api/javax/jms/Topic.html#getTopicName()>`_."
