===================================================
com.github.jcustenborder.kafka.connect.splunk.Event
===================================================

This schema represents the data received from the Splunk listener.


+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Name       | Optional | Schema                                             | Default Value | Documentation                                                                                                                                                                  |
+============+==========+====================================================+===============+================================================================================================================================================================================+
| time       | true     | :ref:`schema-int64`                                |               | The event time.                                                                                                                                                                |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| host       | true     | :ref:`schema-string`                               |               | The host value to assign to the event data. This is typically the hostname of the client from which you're sending data.                                                       |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| source     | true     | :ref:`schema-string`                               |               | The source value to assign to the event data. For example, if you're sending data from an app you're developing, you could set this key to the name of the app.                |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| sourcetype | true     | :ref:`schema-string`                               |               | The sourcetype value to assign to the event data.                                                                                                                              |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| index      | true     | :ref:`schema-string`                               |               | The name of the index by which the event data is to be indexed. The index you specify here must within the list of allowed indexes if the token has the indexes parameter set. |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| event      | true     | :ref:`schema-string`                               |               | This is the event it's self. This is the serialized json form. It could be an object or a string.                                                                              |
+------------+----------+----------------------------------------------------+---------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
