===========================================================================
com.github.jcustenborder.kafka.connect.rabbitmq.BasicProperties.HeaderValue
===========================================================================

Used to store the value of a header value. The `type` field stores the type of the data and the corresponding field to read the data from.

| Name      | Optional | Schema                                             | Default Value | Documentation                                                                                                                             |
+===========+==========+====================================================+===============+===========================================================================================================================================+
| type      | false    | :ref:`schema-string`                               |               | Used to define the type for the HeaderValue. This will define the corresponding field which will contain the value in it's original type. |

| timestamp | true     | :ref:`schema-int64`                                |               | Storage for when the `type` field is set to `timestamp`. Null otherwise.                                                                  |
