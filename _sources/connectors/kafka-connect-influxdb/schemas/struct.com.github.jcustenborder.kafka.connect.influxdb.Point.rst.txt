=====================================================
com.github.jcustenborder.kafka.connect.influxdb.Point
=====================================================

This schema represents the :term:`value` that is written to Kafka.


+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
| Name          | Optional | Schema                                             | Default Value | Documentation                             |
+===============+==========+====================================================+===============+===========================================+
| measurement   | false    | :ref:`schema-string`                               |               | The name of the measurement.              |
+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
| tags          | false    | Map of :ref:`schema-string`, :ref:`schema-string`  |               | Tags associated with the measurement.     |
+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
| timestampNano | false    | :ref:`schema-int64`                                |               | Timestamp in nanosecond precision.        |
+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
| timestamp     | false    | :ref:`schema-int64`                                |               | Timestamp of the message.                 |
+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
| values        | false    | Array of STRUCT                                    |               | The values that were written to InfluxDB. |
+---------------+----------+----------------------------------------------------+---------------+-------------------------------------------+
