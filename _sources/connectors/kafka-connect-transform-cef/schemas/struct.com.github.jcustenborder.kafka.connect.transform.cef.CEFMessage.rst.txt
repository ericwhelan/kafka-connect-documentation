===============================================================
com.github.jcustenborder.kafka.connect.transform.cef.CEFMessage
===============================================================

The parsed representation of a CEF Message.


+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| Name               | Optional | Schema                                             | Default Value | Documentation                                              |
+====================+==========+====================================================+===============+============================================================+
| date               | true     | :ref:`schema-int64`                                |               | The timestamp of the message.                              |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| facility           | true     | :ref:`schema-int32`                                |               | The facility of the message.                               |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| host               | true     | :ref:`schema-string`                               |               | The host of the message.                                   |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| level              | true     | :ref:`schema-int32`                                |               | The level of the syslog message as defined by :rfc:`5424`. |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| message            | false    | :ref:`schema-string`                               |               | Unparsed version of the message.                           |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| cefVersion         | false    | :ref:`schema-int32`                                |               | Version of CEF the message is using.                       |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| deviceVendor       | false    | :ref:`schema-string`                               |               | Vendor of the device that logged the message.              |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| deviceProduct      | false    | :ref:`schema-string`                               |               | The product that logged the message.                       |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| deviceVersion      | false    | :ref:`schema-string`                               |               | The version of the device that is logging the message.     |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| deviceEventClassId | false    | :ref:`schema-string`                               |               | The internal event id for the message.                     |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| name               | false    | :ref:`schema-string`                               |               | Name of the event. This is typically a short description.  |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| severity           | false    | :ref:`schema-string`                               |               | The severity of the message.                               |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
| extensions         | true     | Map of :ref:`schema-string`, :ref:`schema-string`  |               | Key value pairs of any extensions to the message.          |
+--------------------+----------+----------------------------------------------------+---------------+------------------------------------------------------------+
